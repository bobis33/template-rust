# Prerequisites
- Rust 1.93+ ([install](https://rustup.rs/))

# Usage
Run examples: <br>
`cargo run --example [ example_to_run ]`
> run `cargo run --example` to list available examples

Run tests: <br>
`cargo test`

Generate docs: <br>
`cargo doc`

Run benchmarks: <br>
`cargo bench`

Lint project: <br>
`cargo clippy`

Format project: <br>
`cargo fmt`
